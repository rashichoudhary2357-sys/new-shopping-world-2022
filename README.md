# New Shopping World Report 2022

**Author:** Rashi Choudhary  
**Role:** Aspiring Business / Data Analyst

![Dashboard Preview](Screenshot%202025-11-13%20153603.png)

## Project Summary
Interactive Excel dashboard analyzing 2022 sales & orders for an e-commerce business. Includes Orders vs Sales, Channel share, Order status, Top states, Age & gender segmentation, and Sales by gender.

## Files
- `New Shopping World Report 2022.xlsx` — Excel dashboard (download to view)
- `Screenshot 2025-11-13 153603.png` — preview image
- `data/sales_sample.csv` — anonymized sample data (if present)
- `docs/project-report.md` — project walkthrough (coming soon)

## How to view
1. Download the `.xlsx` file and open in Excel 2016+ (PivotTables & slicers supported).  
2. Use slicers (Month / Channel / Category) to filter dashboard.

## Key insights (summary)
- Highest monthly sales: March  
- Top states by sales: Maharashtra, Karnataka, Uttar Pradesh  
- Delivered rate ~92%
